# Section 6.3 Code generation

1. **Go to All > System Definition > Script Includes.** A script is a reusable server-side script that provides logic to define a function or class.

&#x20;

<figure><img src="../.gitbook/assets/script menu.png" alt=""><figcaption></figcaption></figure>

&#x20;

2. Select **New** in the upper right-hand corner.

&#x20;

3. Close any popups that appear, then give the Script Include a Name (e.g., _**\[Your initials] Test Script) and a Description of “My first test script”.**_

&#x20;

4. Replace the default code with the following:

```
//predict assignment group using predictive intelligence classification model
```

&#x20;

5. Now, **hit CMD + Return (Mac) or CTRL + ENTER (Windows)**

&#x20;

<figure><img src="../.gitbook/assets/code gen pick.png" alt=""><figcaption></figcaption></figure>

&#x20;

This feature allows you to use Now Assist to write a custom script based on your instructions.

&#x20;

6\.    When you are finished, click **Submit**

&#x20;

**Congratulations,** you have completed the Now Assist for the Developer persona portion of the lab!
